Restore Copy All Urls in Chrome 138 (Manual Workaround)

As of Chrome version 138, Google has begun disabling extensions that use the older Manifest V2 format—including the popular Copy All Urls extension. If you're seeing a message like “This extension is no longer supported,” it means Chrome has automatically turned it off and blocked it from running.

Fortunately, there's still a workaround that allows you to manually reinstall and use Copy All Urls (and other Manifest V2 extensions) by loading them as unpacked developer extensions. This method works in Chrome 138 and may continue to work in future versions—though it comes with some caveats.

Here's how to bring Copy All Urls back in a few simple steps:

  • Go to https://github.com/felixfbecker/copy-all-urls
  • Click Code > Download ZIP
  • Unzip the folder somewhere on your computer
  • Open chrome://extensions/ in Chrome
  • Toggle Developer Mode ON (top-right)
  • Click Load unpacked
  • Select the folder you just unzipped
Caveats:
  • Chrome will show a warning that “Developer mode is on”
  • You won’t get auto-updates – you'll have to repeat the process if you want new versions
  • These may stop working permanently once Google fully disables V2 in stable builds. Consider switching to Brave or Firefox.
